## 杂记 3 —— 类

__类与对象__

abstract class    
可定义抽象方法与非抽象方法或属性方法（无参方法）

case class    
构造器，成员变量，一些方法

Sealed class    
Sealed 可以修饰 class 与 trait，保证了只能在当前文件中被继承。

object    
定义 apply update 等静态方法

trait    
可以定义抽象方法与紧密联系的相关方法


_注意_

1. class 中调用无参方法，可以不带括号。无参方法看起来像属性

2. 多个重写使用 with 连接

3. 构造器上定义成员变量直接重写属性方法
